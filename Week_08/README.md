##### 2.位运算符
在计算机中：
| 含义 | 运算符 | 示例 |
| :-----: | :-----: | :-----: |
| 左移 | << | 0011=>0110 |
| 右移 | >> | 0110=>0011 |
| 按位或 | &#124; | 0011 &#124; 1011 => 1011 |
| 按位与 | & | 0011 & 1011 => 0011 |
| 按位取反 | ~ | 0011 => 1100 |
| 按位异或（相同为零不同为一） | ^ | 0011 ^ 1011 => 1000 |
